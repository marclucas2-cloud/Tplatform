# PROCEDURE D'URGENCE -- TOUT FERMER

## Methode 1 : Telegram (10 sec)
```
/kill

```
- Ferme TOUTES les positions live
- Annule TOUS les ordres ouverts
- Le paper continue

## Methode 2 : IBKR Mobile (30 sec)
1. Ouvrir l'app IBKR
2. Aller dans Portfolio
3. Selectionner toutes les positions
4. "Close All" -> Confirmer

## Methode 3 : TWS Desktop (1 min)
1. Ouvrir TWS
2. File -> Close All Positions
3. Confirmer
